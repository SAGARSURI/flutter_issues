# flutter_issues

A new Flutter project which showcase all issues in the Flutter repository. You can sort the issues by comments, createdAt, updatedAt. You can filter issues by open, closed, all.

## Getting Started

Libraries used:
* Freezed
* Riverpod
* Mocktail
* build_runner
* http
* fpdart
* very_good_analysis
* flutter_markdown
